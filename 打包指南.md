# 语雀知识库下载工具打包指南

本指南详细介绍如何打包带图形化界面的语雀知识库下载工具。

## 1. 环境准备

### 1.1 安装依赖

使用pnpm安装项目依赖，包括Electron相关依赖：

```powershell
# 安装基础依赖
pnpm install

# 安装Electron相关依赖
pnpm add -D electron electron-builder
```

### 1.2 设置国内镜像源

为了避免网络问题导致的下载失败，设置Electron的镜像源：

```powershell
# 设置Electron镜像源为国内源
$env:ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
$env:ELECTRON_BUILDER_BINARIES_MIRROR="https://npmmirror.com/mirrors/electron-builder-binaries/"
```

## 2. 构建项目

在打包之前，确保先构建项目：

```powershell
pnpm run build
```

## 3. 打包应用

运行electron-builder来打包应用：

```powershell
pnpm run electron:build
```

## 4. 处理常见问题

### 4.1 Electron安装失败

如果遇到Electron安装失败的问题，可以尝试：

```powershell
# 删除现有的electron目录
Remove-Item -Path node_modules\electron -Recurse -Force

# 使用国内镜像源重新安装
pnpm add -D electron --registry=https://registry.npmmirror.com
```

### 4.2 网络连接问题

如果打包过程中遇到网络连接问题：
- 检查网络连接是否稳定
- 尝试使用代理服务器
- 确保已设置国内镜像源

## 5. 打包结果

打包完成后，可执行文件会生成在`dist-electron`目录中：
- `dist-electron/win-unpacked` - 未打包的可执行文件
- `dist-electron/yuque-dl Setup x.x.x.exe` - 安装程序

## 6. 软件授权系统使用流程

### 6.1 授权系统概述

本软件采用RSA非对称加密授权系统，每个授权码与特定设备绑定，确保软件安全使用。

### 6.2 管理员操作（生成授权码）

#### 步骤1：生成RSA密钥对（首次使用）

```powershell
# 运行密钥生成工具
node generate-keys.js
```

执行后会生成两个文件：
- `public-key.pem` - 公钥（用于客户端验证）
- `private-key.pem` - 私钥（用于生成授权码，请妥善保管）

**注意**：请将`public-key.pem`中的内容复制到`electron/license.js`中的`PUBLIC_KEY`变量。

#### 步骤2：为客户生成授权码

```powershell
# 格式：node license-generator.js <机器码> [有效期天数]
# 示例1：生成365天有效期的授权码
node license-generator.js ABCD-EFGH-IJKL-MNOP 365

# 示例2：生成永久授权码
node license-generator.js ABCD-EFGH-IJKL-MNOP 0
```

### 6.3 客户使用流程

#### 步骤1：安装软件

1. 运行安装程序`yuque-dl Setup x.x.x.exe`
2. 按照提示完成安装
3. 安装完成后，软件会自动在安装目录下创建`download`文件夹作为默认下载目录

#### 步骤2：获取机器码

1. 首次打开软件，会显示授权界面
2. 复制界面上显示的**机器码**（格式：XXXX-XXXX-XXXX-XXXX）
3. 将机器码发送给管理员

#### 步骤3：输入授权码

1. 从管理员处获取授权码
2. 在授权界面输入授权码
3. 点击"激活软件"按钮
4. 激活成功后即可使用全部功能

### 6.4 授权文件位置

授权信息保存在用户主目录下的隐藏文件中：
- Windows: `C:\Users\用户名\.yuque-dl-license`
- macOS: `~/.yuque-dl-license`
- Linux: `~/.yuque-dl-license`

**注意**：
- 授权码与设备绑定，更换电脑或重装系统需要重新授权
- 请勿删除授权文件，否则需要重新激活

### 6.5 授权验证

软件每次启动时会自动验证授权：
- 如果授权有效，直接进入主界面
- 如果授权过期或无效，显示授权界面要求重新激活

## 7. 使用方法

### 7.1 下载知识库

1. 打开应用程序
2. 在"下载知识库"标签页中输入相关参数：
   - **语雀知识库URL**：需要下载的知识库地址
   - **下载目录**：文件保存位置（默认为安装目录下的download文件夹）
   - **Cookie Key/Value**：访问私有知识库时需要
3. 点击"开始下载"按钮
4. 下载完成后，可以点击"打开下载目录"查看文件

### 7.2 启动本地服务器

1. 切换到"启动服务器"标签页
2. 选择已下载的知识库目录
3. 设置端口（默认5173）和主机（默认localhost）
4. 点击"启动服务器"按钮
5. 服务器启动后会自动在浏览器中打开预览页面
6. 使用"停止服务器"按钮可以关闭服务

## 8. 注意事项

- 确保网络连接正常，以便下载所需的依赖文件
- 第一次启动应用时可能会较慢，因为需要初始化相关资源
- 如果遇到权限问题，请以管理员身份运行安装程序
- 下载目录默认为安装目录下的`download`文件夹，可以在设置中更改
